**Action.** The system must execute the decomposed sub-goals: query OpenAlex and Semantic Scholar, download papers, parse PDFs, extract figures and tables, write code to re-analyze data, format LaTeX manuscripts, file references in BibTeX, submit to journals via journal-management-system APIs.
